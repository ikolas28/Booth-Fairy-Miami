# Marketing

Future marketing operations area for:

- campaigns
- Instagram promotions
- TikTok weekly content plans
- email outreach
- ad tracking
- lead source reporting

TikTok content is planned inside the CRM only. The marketing agent creates owner-reviewed drafts with hooks, shot lists, captions, hashtags, and CTAs. Booth Fairy Miami should post manually from TikTok unless a future official TikTok Content Posting API integration is approved.

Current TikTok handle: @boothfairymiami. Cadence: weekly.

Instagram campaigns can be approved and published from the CRM after Meta publishing env vars are configured in Vercel. Use campaign notes like:

```
Caption:
Your client-facing caption here.

Media URL: https://public.example.com/photo-or-video.mp4
Media type: reel
```

Only use direct public HTTPS media URLs. Private Google Drive preview links usually will not work for Graph API publishing.
