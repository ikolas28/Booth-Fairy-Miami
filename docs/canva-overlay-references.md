Workflow:

1. Match the client request by event type, theme, color palette, and overlay text.
2. Duplicate the closest Canva reference design.
3. Replace text with the client's exact approved wording.
4. Adjust colors, decorative elements, and safe zones to match the event.
5. Export transparent PNG files at the required photo booth sizes.

These references should be used before outside template marketplaces. Outside marketplaces are only for layout inspiration, not copying.

## Booth Fairy House Style

- Keep the center almost completely clear for guests.
- Use one short event phrase as the main visual text.
- Birthday example style: `Mia's 16th Birthday`, not a long sentence.
- Keep `@boothfairymiami` small and placed on an edge or side.
- Use event/client logos low on the canvas when needed.
- Prefer minimal accents over heavy frames.
- Do not place large panels or blocks over the guest area.
- The overlay should feel light before photos are added.

## Design Research Notes

Photo booth template marketplaces commonly sell editable overlay templates by event category such as wedding, birthday, corporate, holiday, and quince. Most provide editable Canva/PSD-style workflows, transparent PNG exports, and square/portrait/landscape formats. Use these marketplaces to study spacing, safe zones, category styling, and professional typography, but create original Booth Fairy designs.

Useful marketplace/reference sources:

- TemplatesBooth template features: https://help.templatesbooth.com/features/
- TemplatesBooth supported formats: https://help.templatesbooth.com/faqs/software-formats/
- TemplatesBooth no-Photoshop workflow: https://help.templatesbooth.com/faqs/use-templates-without-photoshop/
- Photo Booth Talk template market: https://www.photoboothtalk.com/market/
- Etsy photo booth overlay marketplace examples: https://www.etsy.com/market/photobooth_overlay_templates

Research takeaways:

- Canva editing, Photoshop PSD, booth-software formats, and transparent PNG overlays are standard delivery formats.
- Blank transparent PNG overlays are commonly used when the booth software or workflow adds final text separately.
- Marketplaces separate templates by format and event type, so Booth Fairy should do the same internally.
- The client-facing design should be original, but it can borrow professional layout principles: safe center, readable text, clear hierarchy, and event-appropriate decoration.
- For Booth Fairy's own style, local examples are the strongest reference because they show the brand preference for clean overlays with subtle edge branding.
